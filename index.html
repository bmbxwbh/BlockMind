Features
为 Minecraft 而生的 AI 架构
不是通用聊天机器人套壳，而是从游戏接口到 AI 决策的全链路设计
🧠

三层记忆系统

空间记忆记住建筑保护区，路径记忆缓存成功路线，策略记忆沉淀可复用操作。跨会话持久化，越用越聪明。

🤖

双 Agent 架构

主 Agent 维护对话上下文（~50 Token/次），操作 Agent 无状态执行（<1500 Token/次）。聊天与执行完全隔离。

🛤️

智能导航

记忆驱动寻路：缓存优先 → Baritone 高级寻路 → A* 回退。自动注入建筑保护区，再也不怕炸家。

📝

Skill DSL

YAML 声明式技能定义，L1-L4 四级缓存。常见任务直接走缓存，零 Token 消耗。支持循环、条件、嵌套。

🛡️

五级安全体系

风险评估 → 操作授权 → 紧急接管 → 审计日志 → 安全区限制。高风险操作需玩家确认，致命操作默认禁止。

🛒

Skill 市场

浏览、导入、分享社区技能。支持 YAML 粘贴、URL 导入、批量打包。PR 方式提交到社区仓库。

🔌 Fabric Mod (Java)

  • 13 个 HTTP API 端点 + WebSocket 事件推送
  • 零协议解析，直接调用游戏内部 API
  • 状态采集：方块、实体、背包、世界
  • 动作执行：移动、挖掘、放置、攻击
  • FakePlayer Bot 系统：独立 bot 实体

🐍 Python 后端

  • 双 Agent 架构：聊天/操作上下文隔离
  • 三层记忆：空间/路径/策略，JSON 持久化
  • Skill DSL：YAML 声明式，L1-L4 分级缓存
  • 五级安全：风险评估 + 授权 + 审计
  • WebUI：Miuix Console，暗色主题

Quick Start
三步启动
支持 Linux / macOS / Windows，自动检测环境、安装依赖、下载 MC 服务端。
terminal
# 1. 克隆项目 git clone https://github.com/bmbxwbh/BlockMind.git cd BlockMind # 2. 编辑配置（填入你的 AI API Key） cp config.example.yaml config.yaml vim config.yaml # 3. 一键启动（自动安装依赖 + MC 服务端 + BlockMind + WebUI） chmod +x start.sh ./start.sh # 访问 WebUI: http://localhost:19951
Docker
docker pull ghcr.io/bmbxwbh/blockmind:latest docker run -d -p 19951:19951 --name blockmind \ -v config.yaml:/app/config.yaml:ro \ ghcr.io/bmbxwbh/blockmind:latest
Windows
:: 双击运行 install.bat start_all.bat :: 或命令行 git clone https://github.com/bmbxwbh/BlockMind.git cd BlockMind && start_all.bat
