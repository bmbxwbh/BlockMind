# BlockMind Landing Page — 设计方案

> 设计原则：精致高级，不华丽浮夸。参考 Linear 的暗夜极简美学。

---

## 一、设计理念

**核心气质：**「深夜打开一个精密的工程终端」

- 不是营销轰炸，而是产品自信
- 不是渐变色块，而是光影层次
- 不是花哨动画，而是呼吸般的微动效
- 内容为王，设计为内容服务

**反面清单（绝不做的事）：**
- ❌ 浮夸渐变背景
- ❌ 假数据仪表盘装饰
- ❌ 彩虹配色
- ❌ 过度圆角（不搞圆乎乎的 SaaS 风）
- ❌ 无意义的 emoji 图标墙
- ❌ 假用户评价

---

## 二、设计系统

### 色彩

| 角色 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#08090a` | 近纯黑，带微冷调 |
| 面板背景 | `#0f1011` | 侧栏/卡片底色 |
| 抬升表面 | `#191a1b` | 悬浮卡片 |
| 主文字 | `#f7f8f8` | 近白，不刺眼 |
| 正文 | `#d0d6e0` | 冷银灰 |
| 弱化文字 | `#8a8f98` | 描述/元数据 |
| 极弱文字 | `#62666d` | 时间戳/禁用态 |
| **品牌色** | `#5e6ad2` | 靛蓝紫 — 仅用于 CTA 和交互高亮 |
| 交互高亮 | `#7170ff` | 链接/选中态 |
| 悬停态 | `#828fff` | 更亮的紫 |
| 边框 | `rgba(255,255,255,0.08)` | 半透明白，不抢戏 |
| 极弱边框 | `rgba(255,255,255,0.05)` | 分隔线 |

**色彩策略：** 整页几乎无彩色，只有靛蓝紫一个点缀色。所有层次通过白色透明度递变实现。

### 字体

| 用途 | 字体 | 字重 | 字号 | 字间距 |
|------|------|------|------|--------|
| Hero 标题 | MiSans | 700 | 64px | -1.4px |
| 章节标题 | MiSans | 700 | 40px | -0.88px |
| 小标题 | MiSans | 600 | 20px | -0.24px |
| 正文 | MiSans | 400 | 16px | normal |
| 弱化正文 | MiSans | 400 | 15px | -0.165px |
| 代码 | JetBrains Mono | 400 | 14px | normal |
| 标签 | MiSans | 500 | 12px | normal |

**关键：** MiSans 是小米出品的中西文字体，字形精致、中英文混排自然，非常适合技术产品。

### 间距

- 基础单位：8px
- 章节间距：120px（桌面）/ 80px（手机）
- 卡片内边距：32px
- 按钮内边距：8px 20px

### 圆角

| 元素 | 圆角 |
|------|------|
| 按钮 | 6px |
| 卡片 | 8px |
| 大面板 | 12px |
| 标签 | 9999px（胶囊） |
| 代码块 | 8px |

### 阴影与层次

**核心原则：** 暗色主题不用传统阴影，用**背景亮度递变**表示层次。

| 层级 | 处理 |
|------|------|
| 底层 | `#08090a` 无阴影 |
| 表面 | `rgba(255,255,255,0.02)` 背景 + 边框 |
| 抬升 | `rgba(255,255,255,0.04)` 背景 + 边框 |
| 悬浮 | `rgba(255,255,255,0.05)` 背景 + 边框 |

## 四、动效策略

| 元素 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| 章节入场 | fadeIn + translateY(20px→0) | 600ms | ease |
| 卡片悬停 | 背景亮度渐变 | 200ms | ease |
| 按钮悬停 | 微上移 + 透明度 | 200ms | ease |
| 代码复制 | 文字变化 "点击复制"→"已复制 ✓" | 即时 | — |
| 导航滚动 | 底部边框显现 | 300ms | ease |

**原则：** 只用 opacity + transform，不用 scale/rotate/blur。所有动效都是「呼吸感」而非「表演感」。

## 五、响应式策略

| 断点 | 变化 |
|------|------|
| < 768px | 标题 40px，特性单列，代码块可横滚，section 间距 80px |
| 768-1024px | 特性 2 列，其他不变 |
| > 1024px | 全宽布局，特性 3 列 |

## 六、技术方案

- 单文件 `index.html`，零依赖
- CSS 自定义属性管理设计系统
- IntersectionObserver 驱动滚动入场动画
- `navigator.clipboard` 实现代码复制
- Google Fonts CDN 加载 Inter + JetBrains Mono
- 支持 GitHub Pages 直接部署

## 七、与现有版本的改进

| 维度 | 旧版 | 新版 |
|------|------|------|
| 设计系统 | 自定义色值 | Linear 规范色值 |
| 字体 | 系统字体栈 | MiSans (400/500/600/700) |
| 边框 | 实色边框 | 半透明白边框 |
| 层次 | 阴影 | 背景亮度递变 |
| 品牌色 | 冰蓝 #00D4FF | 靛蓝紫 #5e6ad2（更沉稳） |
| 标题字间距 | normal | 负值压缩（-1.4px） |
| 动效 | 渐入 | 渐入 + 悬停亮度变化 |
| 架构图 | 文字描述 | 等宽字体 ASCII 图 |
